Catan Website Setup
    Windows:
    go to terminal and type this in:
    pip install flask catanatron catanatron_gym sb3-contrib stable-baselines3 gymnasium numpy

    im assuming you unzipped the file

    then go to back to terminal and do cd C:Path to folder location (this folder not the templates one inside)

    then run python app.py and wait for 15-30 seconds so everything will run 
    (if you didn't install the libraries I said above it probably wont work)

    after you see a message about the debugger being active, go to your browser and type in:
    http://localhost:5000 and the game should be present
